Implementing an Algorithm

Algorithms should, ideally, be implemented alongside the creation of the corrsponding parameter file. Implementing an Algorithm involves the following steps:

  1. Create the file: The file should be named <name>.py, where <name> is the name of the algorithm.
    The file can be placed anywhere under the directory Code/algorithm.

  2. Implement the Algorithm class: You should import Algorithm from algorithm.algorithm and create a class that extends it. Your class should implement the following instance methods:
    1. __init__: The constructor. Accepts a BaseParamSet (essentially the same as ParamSet, but you don't specify what section parameters should go in/come from) and uses it to initialize the Algorithm.
    2. run:

      Parameters

      1. app: The Application instance on which to run the experiment.
      2. pset: The ParamSet object for the experiment. Used to look up and (potentially) store parameter values.
      3. timestamp: The timestamp associated with the experiment run. To be passed into the ParamSet's getPath function when saving any additional output files over the course of the experiment.
      4. randId: The random ID associated with the experiment run. Can be passed into the ParamSet's getPath function when saving any additional output files over the course of the experiment.

      Effect

      Runs the Algorithm on the given Application using the given parameters. Should return an object containing values that it wants saved to file.

      At the time of writing, all algorithms output an object with two fields: best_seed_sets and rewards.

    3. refresh: This method should do two things:
      1. It should re-initialize any variables that were initialized using randomness.
      2. It should reset any internal state that it may keep throughout an experiment run back to the starting state.
      If neither are relevant, than it can simply be implemented and do nothing.
  3. Write a 'createInstance' function: This function exists to act as a universal way to create an instance of the Algorithm, it accepts a BaseParamSet object and passes it to the class's constructor, returning the class instance.
